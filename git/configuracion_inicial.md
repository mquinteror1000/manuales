# Configuración inicial de GIT

```bash
git config --list
git config --global user.name "myusername"
git config --global user.email "myemail@gmail.com"

```
** Tanto myusername com myemail@gmail.com deben coincidir con los de la cuenta de github/gitlab para que las contrubuciones puedan acreditarse en el gráfico de contrubuciones
